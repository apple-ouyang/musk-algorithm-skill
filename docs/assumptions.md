# Assumptions

## Execution assumptions

1. Build a single public repository that contains both the research corpus and the skill.
2. Focus the corpus on "The Algorithm" and the most relevant public context, not the entirety of Elon Musk coverage.
3. Convert captured material into Markdown with explicit source attribution.
4. Install the resulting skill into the local agent config tree after the repo version is stable.
5. Commit each major milestone separately and push to GitHub once the repository is ready.
